# Country Info Finder 🌍

A responsive web application that displays country information using the REST Countries API.

## Features
- Dynamic country search
- Region-based filtering
- Lazy loading for images
- Dark/Light mode
- Country details modal
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- REST Countries API

## API
https://restcountries.com/

## Run Project
1. Download the files
2. Open index.html in browser
